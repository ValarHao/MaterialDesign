# MaterialDesign

FloatingActionButton => 继承自ImageView，悬浮于layout之上

Snackbar => 和Toast类似，但多了个可选择的单个的Action。同时还有个特性就是用户可以在它消失时间未到的时候，移掉它

TextInputLayout => 对该控件下的EditText添加一些额外效果

TabLayout => 与ViewPager、Fragment结合实现一种滑动显示界面

NavigationView => 与DrawerLayout结合实现一种侧滑导航抽屉的显示界面

CoordinatorLayout => 包裹Snackbar后使其具有滑动移除的效果

AppBarLayout => 使Toolbar具有上下滑动隐藏、展现的效果

CollapsingToolbarLayout => 使Toolbar具有拉伸、收缩效果

RecyclerView => RecyclerView、CardView、SwipeRefreshLayout结合运用实现下拉刷新

Animation => 波纹效果和动画展现效果
